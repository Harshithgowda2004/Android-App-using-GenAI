# Kabaddi-Arena 🏆

Kabaddi-Arena is an AI-powered Android application developed using Kotlin and Jetpack Compose for tracking and analyzing Kabaddi player performance during local and rural matches.

The application allows users to log raids and tackles in real time, view performance analytics, generate raid heatmaps, and receive AI-based coaching suggestions. The project aims to modernize traditional Kabaddi by introducing professional-level analytics for grassroots athletes.

---

## 🚀 Features

* Live Match Logger
* Raid & Tackle Tracking
* Undo Last Action
* Performance Analytics Dashboard
* Raid Success Percentage Calculation
* Kabaddi Court Heatmap Visualization
* Match Timer
* AI Coach Suggestions
* Match Summary Screen
* Shareable Performance Card UI

## 🎯 Problem Solved

Local Kabaddi tournaments often lack proper player analytics and performance tracking systems. Rural players do not have access to professional statistics to showcase their talent.

Kabaddi-Arena solves this problem by providing:

* Real-time performance tracking
* Raid analytics
* Heatmaps
* AI-based feedback
* Match statistics

This helps players improve their gameplay and gain visibility for higher-level opportunities.
